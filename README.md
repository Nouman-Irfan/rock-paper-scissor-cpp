# 🎮 Rock Paper Scissors Game (C++)

A two-player console-based Rock Paper Scissors game developed in C++.

This project allows two users to play multiple rounds, calculates scores, handles invalid input, and declares the final winner.

---

## 📌 Features

- 👥 Player vs Player mode
- 🔢 Custom number of rounds
- ✅ Input validation (only 1, 2, 3 allowed)
- 🏆 Automatic score calculation
- 🔁 Replay option after game ends
- 📊 Final winner announcement
- 🖥️ Clean formatted console interface

---

## 🎯 Game Rules

Players choose:

1 → Rock  
2 → Paper  
3 → Scissors  

Winning conditions:
- Rock beats Scissors
- Paper beats Rock
- Scissors beats Paper
- Same choice = Draw (No score counted)

---

## 🛠️ Concepts Used

- Loops (`for`)
- Conditional statements (`if-else`)
- Functions (recursive call to `main()` for replay)
- Variables & counters
- Input validation
- Score tracking system
